About FraudGuard AI

FraudGuard AI is a Machine Learning based web application that helps detect fraudulent financial transactions. Users can securely register, login, upload CSV transaction files, analyze transactions using a trained Random Forest model, identify fraudulent transactions, and download fraud reports.

Key Features

🔐 Secure Login

Register and login securely using SQLite database.

📂 CSV Upload

Upload financial transaction datasets for analysis.

🤖 AI Detection

Random Forest Machine Learning model predicts fraudulent transactions.

📊 Dashboard

View transaction summary and fraud statistics.

🚨 Fraud Report

View all fraudulent transactions identified by AI.

⬇ Download Report

Download fraud report as CSV for future investigation.
